# 几个重要的热键

## [tab]键

tab键的主要功能是补全，具有命令补全和文件补全的功能。

* 在命令或者文件后面按两个tab，将对应的命令或者文件列出来

  ```sh
  zhouning@DESKTOP-681HT7R:~$ cal[tab][tab]
  cal       calc.exe  calendar  caller
  ```

* 命令后面按一个tab，实现命令补全

* 文件后面按一个tab，实现文件补全

## [ctrl]-c按键

CTRL+C的组合键的作用是将正在运行的命令中断

## [ctrl]-d按键

ctrl+D组合键的作用是离开文字接口
